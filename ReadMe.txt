Different types of SQL used
1. T-SQL is a syntax for writing SQL and is specific to Microsoft Sql Server.
2. NoSQL is a generic name for databases that favor store semi-structured or non-structured data in a non-relational format.
3. MySql is a specific, open-source and free, relational database engine.
4. PL/SQL
